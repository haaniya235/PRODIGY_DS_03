# PRODIGY_DS_03
This project focuses on building a Decision Tree Classifier using Machine Learning to predict the presence of heart disease based on medical attributes and patient data.

The dataset was preprocessed and analyzed using Python libraries such as Pandas, NumPy, Matplotlib, and Scikit-learn. Data cleaning, feature selection, model training, prediction, and visualization techniques were performed to improve understanding of classification algorithms.

Features:
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Feature and target separation
- Train-test data splitting
- Decision Tree Classifier implementation
- Model accuracy evaluation
- Decision Tree visualization

Technologies Used:
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

Outcome:
The model successfully predicts whether a patient is likely to have heart disease based on the given medical parameters, demonstrating the practical application of Decision Tree algorithms in healthcare analytics.
